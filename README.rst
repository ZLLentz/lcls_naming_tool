`Documentation <https://pcdshub.github.io/lcls_naming_tool/>`_

A tool that checks the form and content of names with respect to the LCLS naming convention.

Requirements
------------

* Python 3.9+

Installation
------------

::

  $ pip install .

Running the Tests
-----------------
::

  $ pytest -v
